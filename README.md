📊 Employee & Project Analytics (SQL Case Study)

A business-focused SQL analytics project simulating real-world HR and project performance analysis.

Overview :

This project analyzes workforce distribution, salary trends, hiring patterns, and project allocation across departments using SQL.
Instead of solving isolated queries, this case study answers structured business questions similar to those asked in product companies and HR analytics teams.

🏢 Business Context

Management needs visibility into:
Workforce distribution across departments
Compensation patterns and salary imbalance
Hiring growth trends
Project allocation efficiency
Risk indicators (low salary, no project allocation)
This project demonstrates how SQL can be used to generate actionable business insights from relational data.

🗄️ Database Schema

Tables Used :
employees
departments
projects
employee_projects

Data Relationships :
Each employee belongs to one department
Each project belongs to one department
Employees can work on multiple projects
