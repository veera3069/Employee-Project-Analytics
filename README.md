📊 Employee & Project Analytics (SQL Case Study)

A business-focused SQL analytics project simulating real-world HR and project performance analysis.

Overview :

This project analyzes workforce distribution, salary trends, hiring patterns, and project allocation across departments using SQL.
Instead of solving isolated queries, this case study answers structured business questions similar to those asked in product companies and HR analytics teams.

🏢 Business Context

Management needs visibility into:
Workforce distribution across departments
Compensation patterns and salary imbalance
Hiring growth trends
Project allocation efficiency
Risk indicators (low salary, no project allocation)
This project demonstrates how SQL can be used to generate actionable business insights from relational data.

🗄️ Database Schema

Tables Used :
employees
departments
projects
employee_projects

Data Relationships :
Each employee belongs to one department
Each project belongs to one department
Employees can work on multiple projects

🛠️ SQL Skills Demonstrated :
Complex Joins
CTEs (Common Table Expressions)
Window Functions
Sub Queries
Analytical Filtering
Aggregations (SUM, COUNT, AVG, MIN, MAX)
NULL Handling

📊 Analytics Performed
🔹 Workforce Analysis
🔹 Compensation Analysis
🔹 Project Performance Analysis
🔹 Advanced Business Insights

📈 Business Insights

🔹 Identifies employees contributing to multiple projects while being the highest-paid member in at least one. This highlights high-impact talent with cross-      project influence and strong compensation positioning.
🔹 Detects projects whose total salary cost exceeds the company’s average project cost. It helps management monitor cost-intensive initiatives and control budget allocation.
🔹 Identifies employees earning above their department’s average salary using window functions. This helps detect premium-paid roles and internal pay differentiation within teams.
🔹 Identifies the department with the highest percentage of employees not assigned to any project. It highlights underutilized teams and supports workforce optimization decisions.
